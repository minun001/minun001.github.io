Connect Auth First

The dashboard shell is ready, but authentication is not configured yet. Add your public Supabase project URL and anon key to /assets/workspace-config.js, then allow the dashboard through the master email or role rule in Supabase.

What stays private

Your password is never stored in this repository. Do not commit service-role keys or any backend secret.

What is public

Supabase URL and anon key are safe for client use. The secure boundary comes from auth rules and row-level security.

Run the SQL setup

Load starter content
